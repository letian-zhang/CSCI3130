Course Description

Topics include introductory assembly language programming, hardware components of digital computers, microprogramming, and memory management. Laboratory exercises involve the development of assembly language codes, logical and functional properties of components from gates to microprocessors. Basic computer architecture, organization, and function of simple computer systems are presented.
Pre-requisite: A grade of "C" or higher in CSCI 2170 is necessary to take this course.

Learning Outcomes

Upon successful completion of this course, a student will be able to:
  • Write assembly language programs using Intel x86-64 and other instruction sets
  • Design and implement various combinatorial and sequential circuits
  • Understand basic computer organization concepts (CPU design and organization)
  • Understand the concepts involved in the memory hierarchy of a computer

Topics Covered

  • Introduction to x86_64 Assembly: gas, system calls, functions (x86_64 Reference)
  • C-library Interfacing and Static Memory Allocation (x86_64,C Reference)
  • Loops, Arithmetic/Logic Ops, Control Ops (x86_64 Reference)
  • Binary/Octal/Hexadecimal, Signed Integers, IEEE Floating Point, Assembly Translation (Ch.2, x86_64 Reference)
  • Computing Fundamentals, Truth Tables, Logic Diagrams, Boolean Expressions, Combinational Circuits (Ch. 3)
  • Simplifying Boolean Expressions, Karnaugh Maps (K-maps, 2-3 variables) (Ch. 3)
  • K-maps (4+ variables), Sum of Products. Product of Sums, Sufficiency Gates (Ch. 3)
  • Complex Combinational Circuits, Half-adder, Full-adder, 3-bit Adder, Shifter, Comparator (Ch. 3)
  • Multiplexer, Decoder, Encoder, RSC Intro, 3-bit ALU (Ch. 3)
  • Basic Sequential Circuits (Flip-flops), Characteristic & Excitation Tables (Ch. 3)
  • Designing Sequential Circuits, Registers, Counters (Ch. 3)
  • Buses, Bus w/MUX+TSBuffer (Ch. 4)
  • Memory Organization (Ch. 6)
  • Introduction to Comp. Org., RSC Registers & Instruction Set
  • Register Transfer Language, RSC Microcode, Purpose of the SC (Ch. 6)
  • Cache Memory (Ch. 6)
  • Virtual Memory (Ch. 6)
